# Gesture Controlled AR Portal

A real-time Computer Vision project built using Python, OpenCV, and MediaPipe.

## Features

- 🖐️ Real-time hand tracking
- ✋ Multi-hand detection
- 🎯 Dynamic polygon generation
- ⚫ Live grayscale portal between hands
- 📷 Real-time webcam processing

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
